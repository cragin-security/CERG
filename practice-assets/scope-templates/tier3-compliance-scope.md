# CERG Engagement Scope — Tier 3: Compliance-to-Operations
## Practice Asset — Not a CERG Corpus Document

---

## In Scope
- Everything in Tier 2 (In Scope), plus:
- Full regulatory operational package implementation (CUI, NERC-CIP, SOX, PCI, ISO, or combination)
- System Security Profile (SCP) per in-scope system
- Evidence library activation with auditor-facing evidence package template
- Quarterly testing cadence activation for each regulator
- Deficiency workflow live test and playbook documentation
- IR adjacent function handoff confirmation
- Service provider (TPSP) compliance oversight integration
- Third maturity assessment with recommendations for Tier 4 transition

## Out of Scope
- Tooling procurement or deployment
- Penetration testing execution (advisory and review only)
- Staff augmentation or interim CISO services

## Deliverables
1. All Tier 2 deliverables
2. Regulatory operational package(s) implemented
3. System Control Profile(s) per system
4. Evidence library with auditor-facing package template
5. Quarterly testing cycle evidence (first full cycle)
6. Deficiency workflow log with exemplar entries
7. IR handshake confirmation record
8. Third maturity assessment
9. Tier 4 transition roadmap

## Duration
180 days. Phased: 90-day structural → 90-day compliance implementation.

## Price Anchor
Phased engagement fee per regulatory package. Quarterly ongoing advisory at fixed quarterly retainer (includes QBR, evidence review, regulatory change monitoring).
