# CERG Engagement Scope — Tier 4: Strategic Partner
## Practice Asset — Not a CERG Corpus Document

---

## In Scope
- Everything in Tier 3 (In Scope), plus:
- Co-design of operating model evolution roadmap (12–24 month horizon)
- Multiple regulatory overlay governance design (concurrent NERC-CIP + CMMC + SOX + PCI)
- Workforce architecture design: role-to-person mapping, grade alignment, hiring pipeline support
- New standard or procedure design through practitioner guild (co-author with client team)
- Practitioner interview and hiring guide activation
- Succession planning and talent review support
- M&A security integration framework
- Practice principal participation in Cyber Oversight Group and board-level briefings
- Dedicated practicing-by-practice advisory for strategic security initiatives (AI, cloud transformation, M&A)
- Quarterly strategic review with written program assessment

## Out of Scope
- Day-to-day security operations (SOC, IR, IAM operations)
- Staff augmentation
- Third-party tool administration

## Deliverables
1. All Tier 3 deliverables
2. Operating model evolution roadmap
3. Multi-regulator governance model
4. Workforce architecture map (role assignments, grade alignment)
5. Guild-authored standards or procedures (co-designed)
6. Strategic initiative advisory documentation
7. Quarterly written program assessment
8. Board-ready briefing materials

## Duration
Ongoing retainer. 12-month renewable commitment with quarterly strategic reviews.

## Price Anchor
Annual retainer with quarterly billing. Strategic advisory day rate for unscheduled work outside scope. Practice principal named as client sponsor.
